Linux Network Services Lab

A practical Linux server administration project built with Rocky Linux Server and Ubuntu Client. This lab demonstrates the configuration, testing, and documentation of essential network services in a local environment.

Apache HTTPS Web Server Active

Configured Services

DNS Server

BIND was configured to resolve local hostnames using forward and reverse lookup zones.

Status: Completed

DHCP Server

Rocky Linux assigns IP addresses and network settings automatically to the Ubuntu client.

Status: Completed

FTP / FTPS Server

vsftpd was configured for file transfer and secured using SSL/TLS encryption.

Status: Completed

Mail Server

Postfix and Dovecot were configured to send and receive email using Thunderbird.

Status: Completed

NFS Server

A shared directory was exported from Rocky and mounted from Ubuntu using NFS.

Status: Completed

SSH Server

Secure remote administration was enabled using OpenSSH from Ubuntu to Rocky.

Status: Completed

Apache Web Server

This web page is hosted using Apache HTTP Server with HTTPS enabled.

Status: Active

Security Monitoring

Honeypot, firewall hardening, and logging will be added in the next stages of the project.

Status: In Progress

Security Highlights

Encrypted Access

HTTPS, FTPS, IMAPS, and SSH were used to secure remote access and data transfer.

Firewall Rules

Only required services and ports were opened using firewalld.

Service Verification

Each service was tested from the Ubuntu client and documented with screenshots.
